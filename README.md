# DBMS-mini-Project
# Employee Management System

This is a **DBMS mini project** to manage employees in a company.  
It allows you to **add, view, update, delete, and search employee records** efficiently.

---

## **Project Idea**

The Employee Management System is designed to **organize and manage employee data**.  
It stores essential details like Employee ID, Name, Age, Email, Department, and Salary.  

### **Purpose**
- Keep track of employees in a company.
- Make searching and updating employee data easier.
- Generate simple reports about employees.

---

## **Features**

- Add new employee records.
- Update existing employee information.
- Delete employee records.
- Search employees by ID, Name, or Department.
- View all employee records in a table format.
- Optional: Generate simple reports (e.g., total salary per department).

---

## **Database Design**

**Database Name:** `company`

**Table:** `employees`

| Column     | Type          | Description         |
|-----------|---------------|-------------------|
| emp_id    | INT (PK)      | Employee ID        |
| emp_name  | VARCHAR(100)  | Employee Name      |
| age       | INT           | Employee Age       |
| email     | VARCHAR(50)   | Employee Email     |
| dept      | VARCHAR(50)   | Department         |
| salary    | FLOAT         | Salary             |
